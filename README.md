# Mangify Backend

## Technologies

-   Flask
-   Sqlalchemy
-   Sqlite3

## Run backend

1. Setup virtual environment:

    ```shell
    python -m venv .venv
    ```

    ```shell
    .venv/Scripts/activate
    ```

    ```shell
    pip install -r requirements.txt
    ```

2. Run server:

    ```shell
    python main.py
    ```
